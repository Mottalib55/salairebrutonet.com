Notre engagement

Rendre l'impôt compréhensible pour tous

Nous croyons que chaque Français devrait pouvoir comprendre comment son impôt est calculé, sans avoir besoin d'un expert-comptable.

Notre Mission

La fiscalité française est souvent perçue comme complexe et opaque. Barème progressif, quotient familial, déductions, réductions... Ces termes peuvent sembler intimidants.

Notre objectif est simple : démystifier l'impôt sur le revenu et donner à chacun les outils pour comprendre sa situation fiscale en toute autonomie.

Pourquoi ce projet ?

Le constat

Trop de Français découvrent le montant de leur impôt sans vraiment comprendre comment il a été calculé. Les simulateurs existants donnent un chiffre, mais rarement les explications.

Notre solution

Un simulateur qui ne se contente pas de calculer, mais qui explique. Chaque tranche, chaque déduction, chaque économie est détaillée pour que vous compreniez vraiment.

Ce que nous croyons

L'impôt n'est pas un ennemi. C'est un outil de solidarité nationale. Mais pour l'accepter, il faut d'abord le comprendre.

La transparence crée la confiance. Quand vous voyez exactement comment chaque euro est calculé, vous reprenez le contrôle.

L'optimisation fiscale n'est pas réservée aux riches. Le PER, les dons, les réductions d'impôt... Ces outils sont accessibles à tous. Encore faut-il les connaître.

La simplicité est une forme de respect. Si un outil est compliqué, c'est qu'il n'a pas été assez travaillé.

Nos engagements

Transparence totale

Chaque calcul est expliqué. Pas de boîte noire, vous voyez exactement d'où vient chaque euro.

100% Confidentialité
