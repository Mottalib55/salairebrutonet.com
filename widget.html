Widget Calculateur Brut Net

Intégrez gratuitement notre calculateur salaire brut en net sur votre site web. Le widget est responsive, léger et se met à jour automatiquement avec les taux 2026.

Aperçu

Code d'intégration

Copiez et collez ce code HTML dans votre site :

<iframe
  src="https://salairebrutonet.com/embed/"
  width="100%"
  height="350"
  frameborder="0"
  style="border:none; border-radius:12px; max-width:420px;"
  title="Calculateur Salaire Brut Net 2026"
  loading="lazy"></iframe>

Conditions d'utilisation

  • Le widget est entièrement gratuit pour un usage personnel ou professionnel.
  • Le lien vers SalaireBrutNet.com doit rester visible dans le widget.
  • Les données sont fournies à titre indicatif et ne remplacent pas un bulletin de paie officiel.
